## TASK 5 — AC5, AC6 coverage-confidence gate

**Files:** `src/kb/query/engine.py`, `tests/test_cycle14_coverage_gate.py`
**Change:**
1. `vector_search` inside `search_pages` attaches `vector_similarity: float` to each returned hit (per Q11 amendment). The same value ALREADY exists as `score = round(1.0 / (1.0 + dist), 4)` at line 144 — additionally stash to `r["vector_similarity"] = r["score"]` BEFORE the results go into RRF.
2. Extend `search_telemetry` schema: `search_telemetry["vector_scores_by_id"] = {pid: sim}` populated inside `vector_search` for each kept result. ADDITIVE, keyword-only mutation.
3. `query_wiki` after `search_pages` returns:
   - `vector_scores = search_telemetry.get("vector_scores_by_id", {})`
   - `ctx_vector_scores = [vector_scores[pid] for pid in ctx["context_pages"] if pid in vector_scores]`
   - `coverage_confidence = (sum(ctx_vector_scores) / len(ctx_vector_scores)) if ctx_vector_scores else None`
4. Gate: if `coverage_confidence is not None and coverage_confidence < QUERY_COVERAGE_CONFIDENCE_THRESHOLD`:
   - Skip `call_llm`.
   - `answer = "Coverage {score:.2f} below threshold {threshold:.2f}. Consider rephrasing or widening the query."`
   - `result_dict["low_confidence"] = True`
   - `result_dict["advisory"] = "..."` (same fixed template, no user-question interpolation; per Q2/Q13).
5. Always include `coverage_confidence` in return dict (float or None).

**Test:** 7 cases:
- Coverage above threshold → no `low_confidence` / `advisory` keys BUT `coverage_confidence` present as float.
- Coverage below threshold → both keys set + `coverage_confidence` float + `answer` is refusal template.
- No vector hits packed (BM25-only) → `coverage_confidence is None`, no `low_confidence` key.
- Gate does NOT call the LLM when triggered (spy on `call_llm` shouldn't be invoked).
- Advisory string contains no user-question text (T5): question = `<script>alert(1)</script>` → advisory body has no `<script>`.
- Integration: `query_wiki` result shape backward-compatible for above-threshold callers.
- Empty context_pages (no pages retrieved) → `coverage_confidence is None`.

**Criteria:** AC5, AC6.
**Threat:** T5 (advisory no-echo).
