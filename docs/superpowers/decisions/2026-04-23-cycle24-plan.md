## Ordering rationale

AC14 (sentinel section-span search) MUST land BEFORE AC1 (inline evidence at first write). Rationale: AC1 emits `SENTINEL` in every new page's first write. Without AC14's span-limited search, an attacker planting `<!-- evidence-trail:begin -->` inside a source body causes future `append_evidence_trail` calls to inject under the attacker-planted sentinel. Shipping AC1 without AC14 first would widen an existing latent vulnerability. Per CONDITION 1 + 9, AC1 and AC14 MUST both ship in this cycle; ordering ensures no mid-cycle window.

### TASK 1 — `evidence.py` + AC14/AC15 regression tests (1 commit)

**Files:** `src/kb/ingest/evidence.py`, `tests/test_cycle24_evidence_sentinel_anchored.py` (new).

**Production change (AC14):**
- Add `render_initial_evidence_trail(source_ref: str, action: str, entry_date: str | None = None) -> str` helper that returns `f"\n## Evidence Trail\n{SENTINEL}\n{format_evidence_entry(d, source_ref, action)}\n"`. Used by TASK 3 `pipeline.py`. Reuses existing `format_evidence_entry` for pipe-neutralization.
- Rewrite `append_evidence_trail`'s sentinel search to SPAN-LIMITED:
  1. `header_match = re.search(r"^## Evidence Trail[ \t]*\r?\n", content, re.MULTILINE)`
  2. If header present: find `section_end` via `re.search(r"^## ", content[header_match.end():], re.MULTILINE)` → absolute offset `header_match.end() + section_end.start()` OR `len(content)` if no next section.
  3. Search for `SENTINEL` ONLY within `content[header_match.end():section_end]`.
  4. Four fallback branches per CONDITION 9:
     - Header present + sentinel-in-span → insert after sentinel (existing behavior preserved).
     - Header present + no-sentinel-in-span → plant sentinel at header end + insert entry (migration case).
     - No header at all + no body sentinel → create fresh section at EOF with sentinel.
     - No header + body sentinel present (attacker scenario) → IGNORE body sentinel; create fresh section at EOF (this falls out of the span-search naturally).

3. `test_crlf_header_with_trailing_whitespace_matched` — header `## Evidence Trail  \r\n`; regex `[ \t]*\r?\n` matches; entry lands correctly.
4. `test_sentinel_only_no_header_creates_fresh_section` — body contains only `<!-- evidence-trail:begin -->` (no header); call `append_evidence_trail`; assert fresh `## Evidence Trail\n<!-- evidence-trail:begin -->\n<entry>\n` appended at EOF; body sentinel unchanged.

  ```python
  def rebuild_vector_index(wiki_dir: Path, force: bool = False) -> bool:
      # AC6: unlink stale .tmp at first statement (before all gates)
      vec_path = _vec_db_path(wiki_dir)
      tmp_path = vec_path.parent / (vec_path.name + ".tmp")
      try:
          tmp_path.unlink(missing_ok=True)
      except OSError:
          pass  # best-effort: if unlink fails, subsequent sqlite3 ops will surface it

      if not _hybrid_available:
          return False
      if not force and not _is_rebuild_needed(wiki_dir):
          return False

      with _rebuild_lock:
          if not force and not _is_rebuild_needed(wiki_dir):
              return False

          from kb.utils.pages import load_all_pages
          pages = load_all_pages(wiki_dir=wiki_dir)
          vec_path.parent.mkdir(parents=True, exist_ok=True)

          # AC5 + AC8: build to tmp, replace atomic, cleanup on crash
          try:
              if not pages:
                  VectorIndex(vec_path).build([], db_path=tmp_path)
              else:
                  texts = [page.get("content", "") for page in pages]
                  model = _get_model()
                  embeddings_np = model.encode(texts)
                  entries = [(pages[i]["id"], embeddings_np[i]) for i in range(len(pages))]
                  VectorIndex(vec_path).build(entries, db_path=tmp_path)

              # CONDITION 2: pop + close BEFORE os.replace
              with _index_cache_lock:
                  popped = _index_cache.pop(str(vec_path), None)
              if popped is not None and popped._conn is not None:
                  try:
                      popped._conn.close()
                  except Exception:
                      pass
                  popped._conn = None

              os.replace(str(tmp_path), str(vec_path))
          except Exception:
              # Clean-slate on crash
              try:
                  tmp_path.unlink(missing_ok=True)
              except OSError:
                  pass
              raise

          entry_count = 0 if not pages else len(entries)
          logger.info("Vector index rebuilt: %s (%d entries)", vec_path, entry_count)
          return True
  ```
