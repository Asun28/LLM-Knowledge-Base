## 3. Threats (numbered — testable pass/fail)

### T1 — Absolute-path leak via `Error[partial]:` MCP return string (kb_ingest_content)
**Surface:** `src/kb/mcp/core.py:761-764` — `f"Error[partial]: write to {_rel(file_path)} failed ({write_err}); ..."` interpolates raw `write_err`.
**Threat:** Windows `OSError.__str__()` returns `"[WinError 5] Access is denied: 'D:\\Projects\\llm-wiki-flywheel\\raw\\articles\\foo.md'"`. Cycle-32 AC3 routes `Error[partial]:` strings to CLI stderr.
**Mitigated by:** AC1 (swap `{write_err}` → `{_sanitize_error_str(write_err, file_path)}`).

### T2 — Absolute-path leak via `Error[partial]:` MCP return string (kb_save_source)
**Surface:** `src/kb/mcp/core.py:880-883`.
**Mitigated by:** AC2.

### T3 — Absolute-path leak via captured `logger.warning` records
**Surface:**
- `src/kb/mcp/core.py:756-760` (kb_ingest_content)
- `src/kb/mcp/core.py:875-879` (kb_save_source)
- `src/kb/mcp/core.py:280` (kb_query.save_as).

**Mitigated by:** AC3 + AC4.
**Caplog stability note:** `pytest-xdist` test isolation requires `caplog.set_level(logging.WARNING, logger="kb.mcp.core")` inside the test.

### T4 — Test fixture passes `OSError` without `filename=` argument → false-positive pass
**Threat:** `OSError("Access is denied")` without third arg has `filename=None` and `__str__()` produces just `"Access is denied"` — no path literal. Test passes on BOTH patched and reverted code.
**Mitigated by:** AC5's exact constructor — `OSError(13, "Access is denied", "D:\\\\Projects\\\\test\\\\fake.md")` (3-arg form).

### T5 — Asymmetric redaction: log line leaks, return string is clean (kb_query.save_as)
**Surface:** `src/kb/mcp/core.py:279-281` — line 280 logs raw `exc`, line 281 returns sanitised.
**Mitigated by:** AC4.

### T6 — `_sanitize_error_str` exception-attribute regex misses non-Path filename
**Surface:** `src/kb/utils/sanitize.py:68-76` — bytes-filename edge case.
**Status:** Out of scope; helper is already cycle-18 verified.

### T7 — Index-file dedup contract regression (silent re-ingest duplication)
**Surface:** `src/kb/ingest/pipeline.py:777` and `pipeline.py:818`.
**Mitigated by:** AC6 (docstring) + AC7 (call-twice idempotency test) + AC8 (crash-then-re-ingest scenario).
**Behaviour-over-signature check:** test must call the actual function and read the file content, not `inspect.getsource()`.

### T8 — `_update_sources_mapping` merge-branch regression
**Surface:** `pipeline.py:781-791` (merge branch).
**Mitigated by:** AC8 step (e)-(f) — pin merge-on-new-pages behaviour.

### T9 — Test-file pollution of real `wiki/` or `raw/`
**Surface:** new test files.
**Mitigated by:** explicit `wiki_dir=tmp_wiki` keyword on `_update_*` calls.

### T10 — Caplog-handler attachment asymmetry under pytest-xdist
**Mitigated by:** explicit `caplog.set_level(logging.WARNING, logger="kb.mcp.core")` at top of each test.

### T11 — Stale CHANGELOG narrative claims redaction without test coverage
**Mitigated by:** Step-11 verification checklist including revert-fail check from cycle-24 L4.

## 4. Authn/authz needs

**No new authn/authz surfaces.** The redaction is **defence-in-depth against accidental leakage to lower-trust surfaces** (CI logs uploaded to a public artifact store, screen-recorded demos, IDE captured stderr in shared screen-share), not against a privileged adversary.

## 5. Logging / audit requirements

### Must NOT appear after AC1-AC5 ship

Under a forced `OSError(13, "Access is denied", "D:\\Projects\\llm-wiki-flywheel\\raw\\articles\\foo.md")`:

- **MCP return string (B1, B2)** for `kb_ingest_content`, `kb_save_source`: NO `D:\` literal, NO `D:/Projects/` literal, NO `/home/`, `/Users/`, `/tmp/`, `/opt/`, `/var/`, `/srv/`, `/mnt/`, `/root/` literal anywhere.
- **`caplog.records` text (B3)** at WARNING level for `kb.mcp.core`: same constraint.
- **Static `[WinError N]` prefix or POSIX `[Errno N]` prefix:** ALLOWED.
- **`<path>` regex-substituted placeholder:** ALLOWED.
- **`raw/articles/foo.md` (relative form):** ALLOWED.

### Must continue to appear (do not over-redact)

- errno / strerror text (`[WinError 5] Access is denied`).
- Relative path (`raw/articles/foo.md`).
- `wiki/log.md` (UNCHANGED).
- `.data/ingest_log.jsonl` (UNCHANGED).
