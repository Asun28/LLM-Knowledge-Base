## TASK 9: `capture.py` two-pass write + hidden-temp reservation (AC10 CRITICAL)

**Files:** `src/kb/capture.py` (`_write_item_files` 555-651), `tests/test_cycle17_capture_two_pass.py` (NEW)
**Change:** Restructure `_write_item_files` from current per-item-commit (Phase A resolve + Phase B alongside compute + Phase C per-item write with retry) into two-pass all-or-nothing:

```python
def _write_item_files(items, provenance, captured_at, *, captures_dir=None):
    if not items:
        return [], None
    _captures_dir = captures_dir if captures_dir is not None else CAPTURES_DIR
    _captures_dir.mkdir(parents=True, exist_ok=True)

    # Phase 1 — reserve hidden temp files for all N slugs atomically.
    with os.scandir(_captures_dir) as it:
        existing = {entry.name[:-3] for entry in it if entry.is_file() and entry.name.endswith(".md")}
    reservations: list[tuple[str, Path, dict]] = []  # (slug, temp_path, item)
    try:
        for item in items:
            slug, temp_path = _reserve_slug(item, existing, _captures_dir)  # retries up to 10x
            reservations.append((slug, temp_path, item))
            existing.add(slug)
    except (CaptureError, OSError) as e:
        _rollback_reservations(reservations)  # unlink all previously-reserved .reserving files
        return [], f"Error: reservation failed: {e}"

    finalized_slugs = [r[0] for r in reservations]

    # Phase 2 — compute alongside_for from FINALIZED slugs (closes the bug).
    alongside_for = [
        [s for j, s in enumerate(finalized_slugs) if j != i]
        for i in range(len(reservations))
    ]

    # Phase 3 — write real content then os.replace temp → final.
    written: list[CaptureItem] = []
    try:
        for i, (slug, temp_path, item) in enumerate(reservations):
            markdown = _render_markdown(
                item=item,
                captured_alongside=alongside_for[i],
                provenance=provenance,
                captured_at=captured_at,
            )
            final_path = _captures_dir / f"{slug}.md"
            if not _is_path_within_captures(final_path, base_dir=_captures_dir):
                _rollback_reservations(reservations[i:])
                return written, f"Error: slug escapes captures dir: {slug!r}"
            temp_path.write_text(markdown, encoding="utf-8")
            os.replace(temp_path, final_path)
            written.append(CaptureItem(slug=slug, path=final_path, title=item["title"],
                                       kind=item["kind"], body_chars=len(item["body"])))
    except OSError as e:
        # plan-gate BLOCKER 2: FULL rollback — unlink committed .md files too.
        _rollback_finalized(written, _captures_dir)  # unlink committed .md files
        _rollback_reservations(reservations[i:])  # unlink remaining temp reservations
        return [], f"Error: write failed on item {i}: {e}"
    return written, None
```

Helpers:
- `_reserve_slug(item, existing, captures_dir) -> (slug, temp_path)`: uses `_build_slug` + `os.open(..., O_CREAT | O_EXCL | O_WRONLY)` on `.{slug}.reserving` path, retries on `FileExistsError` up to 10 times with `_build_slug`-computed alternate.
- `_rollback_reservations(reservations)`: for `(slug, temp_path, _)` in list, `temp_path.unlink(missing_ok=True)`.
- `_rollback_finalized(written, captures_dir)`: for `item in written`, `item.path.unlink(missing_ok=True)`. (plan-gate BLOCKER 2: all-or-nothing requires unlinking finalized `.md` files on Phase-3 mid-batch failure.)

**Test (`tests/test_cycle17_capture_two_pass.py`):**
- `test_alongside_for_uses_finalized_slugs_under_collision`: monkeypatch `_build_slug` to simulate Phase-1 collision where initial slug becomes alternate on retry; assert `captured_alongside` in the written file references the FINALIZED peer slug, not the initial guess.
- `test_partial_reservation_rollback`: monkeypatch `os.open` to succeed for items 0-2 and raise `OSError` on item 3; assert all 4 `.reserving` temp files are unlinked AND return value is `([], error_msg)`.
- `test_phase3_failure_rolls_back_all` (plan-gate BLOCKER 2 fix — was `test_phase3_failure_preserves_committed_items`): monkeypatch `os.replace` to succeed on items 0-1 and raise `OSError` on item 2; assert NO `.md` files remain in `captures_dir` (items 0-1 ALSO unlinked — all-or-nothing), NO `.reserving` temp files remain, return value is `([], error_msg)`.
- `test_hidden_temp_not_matched_by_md_glob`: reserve a slug, scan `captures_dir` via `glob("*.md")`, assert temp file NOT matched (starts with `.`, ends with `.reserving`).
**Criteria:** AC10.
**Threat:** T4.
