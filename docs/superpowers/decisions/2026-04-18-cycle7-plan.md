## Red-flag self-checks before each commit

1. No `re.findall(source)` or `inspect.getsource(module)` asserts in the new test file (grep pre-commit).
2. No `f"Error: ... {e}"` regression sites introduced by other ACs' changes (grep post-implementation).
3. Every new function added to a module has at least one orchestrator call site (no orphan helpers).
4. Every optional `pages=None` kwarg default is honoured ‚Äî no call breaks when omitted.
5. `ruff format` run AFTER every Edit in a task, re-Read file before next Edit.
6. Commit messages follow `<type>(<scope>): <AC ref or summary>` pattern.
