- [ ] **Fix 3.10: `_mask_code_blocks` placeholder collision** (`linker.py:24-35`)
Use per-call UUID prefix for placeholders. **The function signatures change** — all callers must be updated simultaneously:
```python
import uuid

def _mask_code_blocks(text: str) -> tuple[str, list[str], str]:
    masked: list[str] = []
    prefix = uuid.uuid4().hex[:8]
    def _replace(m):
        masked.append(m.group(0))
        return f"\x00{prefix}{len(masked) - 1}\x00"
    return _CODE_MASK_RE.sub(_replace, text), masked, prefix

def _unmask_code_blocks(text: str, masked: list[str], prefix: str) -> str:
    for i, code in enumerate(masked):
        text = text.replace(f"\x00{prefix}{i}\x00", code)
    return text
```

- Line 167: `body, masked_code = _mask_code_blocks(body)` → `body, masked_code, prefix = _mask_code_blocks(body)`
- Line 196: `new_body = _unmask_code_blocks(new_body, masked_code)` → `new_body = _unmask_code_blocks(new_body, masked_code, prefix)`
